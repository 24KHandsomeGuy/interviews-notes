## 1.InfoQ

InfoQ(ueue)基本上就是日刷，可以看下同行都在做啥，不至于闭门造车。

InfoQ提供新闻、文章、视频演讲和采访等资讯服务。

**InfoQ有各种语言版本，最重要的是有中文版的，内容相当有深度，非常有益于把握最新的技术动态。**

<https://www.infoq.cn/>
